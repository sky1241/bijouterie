# 📸 Liste de prise de vue — Colin Philippe

> À imprimer / cocher pendant le shooting. Objectif : remplacer les images d'illustration du site par les vraies pièces et l'atelier.

## Réglages généraux (à respecter pour tout)
- [ ] **Format portrait** (vertical, ~4:5) — les cases du site sont verticales.
- [ ] **Fond neutre et propre** : blanc, gris clair, ou velours sombre. Pas de fouillis derrière.
- [ ] **Lumière douce** : près d'une fenêtre (lumière du jour) ou softbox. **Pas de flash direct** (reflets sur l'or/les pierres).
- [ ] **Netteté** : pièce bien au point. Nettoyer les pièces avant (traces de doigts).
- [ ] Prendre **2-3 angles** par pièce (de face + léger 3/4) → on choisit la meilleure.
- [ ] Un iPhone récent (ou appareil photo) suffit. Mode portrait OK pour les pièces.
